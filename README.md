### How do I get set up?

- Be sure you have node.js installed.
- Make sure you are at the correct directory named ./medibank-coding-challenge or root folder.
- You will need to run "npm install" or "yarn" on your terminal depending on your preference.
- To view the history progress of my commits, run "git log" on your terminal.
- To run test you will need to run "npm run test" or "yarn test" on your terminal depending on your preference.
- Run "npm start" or "yarn start" to view the web application on my work.

## Requirements

You need to:

- Write some code to consume the json hosted on the above web service.
- Output a list of all the cats in alphabetical order under a heading of the gender of their owner.
- Output must be presentable on a web browser.
- Submissions will only be accepted via GitHub or Bitbucket.

Please note:

- It can be written in any language you like.
- Use any libraries/frameworks/SDKs you choose.
- Use industry best practices.
- Use the code to showcase your skill and what you value in a software application.
